# FAQ

## What is this project?

Game Hub Ultimate is an open-source directory for organizing game guides, patch notes, walkthrough references, modding resources, and community links.

## Can I submit a guide for my game?

Yes. Follow the template in `templates/game-page.md` and open a pull request.

## Does this project host game files or mods?

No. This project is a documentation directory. It does not host game files, mods, or executable downloads.

## How do I report outdated content?

Open an issue with the page path and describe what needs updating.

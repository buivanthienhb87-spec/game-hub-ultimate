# Recommended Game Development and Player Resource Websites

This page collects official and reliable websites that can help maintainers verify game information, document updates, and guide players to safer resources. The list avoids promotional links and focuses on stable references.

| Website | Recommended Use |
|---|---|
| [Steam News Hub](https://store.steampowered.com/news/) | Official Steam update and announcement pages for PC games. |
| [PlayStation Blog](https://blog.playstation.com/) | Official PlayStation announcements, release notes, and platform news. |
| [Xbox Wire](https://news.xbox.com/) | Official Xbox news, game updates, and ecosystem announcements. |
| [Nintendo News](https://www.nintendo.com/us/whatsnew/) | Official Nintendo updates and game announcements. |
| [itch.io Developer Community](https://itch.io/community) | Community resources for indie game developers and players. |
| [PCGamingWiki](https://www.pcgamingwiki.com/) | Community-maintained reference for PC game fixes and configuration notes. |
| [MobyGames](https://www.mobygames.com/) | Long-running database for game credits, release information, and historical references. |
| [IGDB](https://www.igdb.com/) | Structured video game database useful for metadata cross-checking. |
| [Unity Learn](https://learn.unity.com/) | Official Unity tutorials and learning resources. |
| [Unreal Engine Documentation](https://dev.epicgames.com/documentation/) | Official Unreal Engine technical documentation. |

## Review Policy

Recommended websites should be reviewed periodically. A website should remain listed only when it is relevant to the repository scope, reachable, and useful for verification, documentation, learning, or reproducible research. Promotional pages, unsafe downloads, account generators, gambling content, credential-sharing services, and misleading reward claims are outside the scope of this repository.

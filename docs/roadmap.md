# Roadmap

The roadmap focuses on building a trustworthy game resource directory. Near-term work includes more guide templates, a static index generator, source review fields, and better multilingual structure.

| Milestone | Outcome |
|---|---|
| Content schema | Stable fields for resource entries. |
| Static index | A generated browsing page from JSON data. |
| Review workflow | Clear stale-content review process. |
| Localization | Reviewed translations for key templates. |
